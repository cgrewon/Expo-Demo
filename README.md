# ExpoTest

This is Expo test project.

##Tech stacks:



- Expo
- Typescript
- redux
- redux-persist
- redux-thunk
- [React Native Elements] (https://reactnativeelements.com/)
- graphql
- @apollo/client



## Install packages:
```shell
$ yarn
```

## Run ios:

```shell
$ yarn ios
```

## Run android:

```shell
$ yarn android
```
